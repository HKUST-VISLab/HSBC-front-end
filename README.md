# hsbc-front-end

[![Greenkeeper badge](https://badges.greenkeeper.io/HKUST-VISLab/hsbc-front-end.svg)](https://greenkeeper.io/)

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
